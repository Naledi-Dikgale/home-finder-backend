# 📖 [Home-finder-backend] <a name="about-project"></a>

**[Home-finder-backend]** is a backend rails api that provides service to the front-end react app to fetch data for rental and for-sale homes.

## 💻 Getting Started <a name="getting-started"></a>

To get a local copy up and running, follow these steps.

### Prerequisites

In order to run this project you need:

- Install Ruby
- Install Rails
- Install Postgresql

### Setup

Clone this repository to your desired folder:

```sh
  cd home-finder_backend
  git clone https://github.com/Naledi-Dikgale/home-finder-backend.git
```

### Install

Install this project with:

```sh
  cd home-finder-backend
  bundle install
```

```sh
  gem install --no-document rubocop -v '>= 1.0, < 2.0'
```

### Usage

To run the project, execute the following command:

```sh
  rails db:create
  rails db:migrate
  rails db:seed
  rails server
```

### Run tests

To run tests, run the following command:

```sh
  rspec
```
